# SQLAlchemy

# Introduction

- ORM 為 Object Relational Mapper 的縮寫
- 透過 Python 物件建立與 Table 之間的映射關係
- 使用 Python 語法操作資料庫，而非 SQL 語法
- SQLAlchemy 為一種 ORM 框架的工具
- 官方文件
    
    [https://www.sqlalchemy.org/](https://www.sqlalchemy.org/)
